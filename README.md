# 🤖 MINI IA - QR Code Generator

Projet Python combinant une mini intelligence artificielle et un générateur de QR codes.

## 🚀 Fonctionnalités

* Génération de QR codes à partir de texte ou d’URL
* Assistant simple en Python
* Sauvegarde des données (memory.json)
* Code modulaire et évolutif

## 🛠️ Technologies utilisées

* Python 3
* Bibliothèque `qrcode`
* JSON (stockage de données)

## 📦 Installation

1. Cloner le projet :

```
git clone https://github.com/marcellin599/Mini_IA-Qrcode-Python.git
cd Mini_IA-Qrcode-Python
```

2. Installer les dépendances :

```
pip install qrcode[pil]
```

## ▶️ Utilisation

Lancer le projet :

```
python app.py
```

Ou selon ton système :

```
python3 app.py
```

## 📁 Structure du projet

* `app.py` : point d’entrée principal
* `assistant.py` : logique de l’assistant
* `memory.json` : stockage des données
* `build/` et `dist/` : fichiers générés (optionnel)

## ⚠️ Remarques

* Assurez-vous d’avoir Python installé
* Les fichiers `build/` et `dist/` peuvent être ignorés
* Projet en cours d’amélioration 🚧
